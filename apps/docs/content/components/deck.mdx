## Features

- Tinder-like swipe gestures (left/right)
- Smooth animations using Motion
- Composable API - children become swipeable cards
- Customizable swipe threshold and stack appearance
- Callbacks for swipe events
- Stack visualization with perspective and scaling
- Touch and mouse drag support
- Automatic card progression
