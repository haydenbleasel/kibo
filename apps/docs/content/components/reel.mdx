## Features

- Full-height, 9:16 aspect ratio container optimized for vertical video
- Progress indicators showing current and total reels
- Automatic video playback with seamless transitions
- Support for both video and image content
- Customizable controls for play/pause, mute/unmute, and navigation
- Touch-friendly navigation by tapping left/right sides
- Composable architecture for custom overlays and content
- Header and footer sections for metadata display
- Social media-style interactions support
- Keyboard accessibility for all controls
- Auto-advance to next reel when current finishes
- Responsive design that scales with viewport height

## API Reference

### Reel

The root component that provides context and container for all reel elements.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultIndex` | `number` | `0` | Initial reel index to display |
| `autoPlay` | `boolean` | `true` | Whether to automatically play videos |
| `defaultMuted` | `boolean` | `true` | Whether videos start muted |
| `onIndexChange` | `(index: number) => void` | - | Callback when reel index changes |

### ReelContent

Container for reel items that handles transitions and animations.

### ReelItem

Individual reel container that wraps video or image content.

### ReelVideo

Video component with automatic playback control and progress tracking.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `onLoadedMetadata` | `(duration: number) => void` | - | Callback when video metadata loads |
| `onTimeUpdate` | `(currentTime: number) => void` | - | Callback on video time update |
| `onEnded` | `() => void` | - | Callback when video ends |

### ReelImage

Image component with automatic duration and progress simulation.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `duration` | `number` | `5` | Display duration in seconds |
| `alt` | `string` | - | Alternative text for accessibility |

### ReelProgress

Progress bar showing current position and total reels.

### ReelControls

Control buttons for play/pause, mute/unmute, and navigation.

### ReelNavigation

Invisible touch areas for tap-to-navigate functionality.

### ReelHeader

Top overlay section for metadata like author info.

### ReelFooter

Bottom overlay section for descriptions and interactions.

### ReelOverlay

General purpose overlay container for custom content.
