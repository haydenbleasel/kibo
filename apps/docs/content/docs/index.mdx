## Introduction

Roadmap UI is a collection of accessible, customizable, performant and open source components for building interactive roadmaps.

These components are designed to be used as variants of the same data model, allowing you to easily switch between them.

The components are inspired by best-in-class tools like [Linear](https://linear.app/).
