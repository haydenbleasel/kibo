## Features

- Displays 5 consecutive days in a compact horizontal layout
- Navigation with chevron buttons to move between date ranges
- Fully composable with separate components for each part
- Supports controlled and uncontrolled usage
- Highlights today's date and selected date
- Built with accessibility in mind
- Customizable styling through className props
- Context-based state management between components
