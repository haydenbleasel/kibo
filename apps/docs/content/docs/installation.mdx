## Prerequisites

Before installing Roadmap UI, make sure you have the following prerequisites:

- [Node.js](https://nodejs.org/en/download/), version 18 or later
- [pnpm](https://pnpm.io/installation), version 9 or later
- A [React](https://reactjs.org/docs/getting-started.html) project, version 18 or later

## Installation

Install Roadmap UI using your favorite package manager. Choose one of the following methods:

```package-install
npm i roadmap-ui
```

## Usage

After installing Roadmap UI, you can start using its components in your React application.

For example, check out the [Gantt Chart](/docs/gantt) component.
