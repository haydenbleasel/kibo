## Getting Started

Install Roadmap UI using your favorite package manager.

<Tabs items={['bun', 'pnpm', 'yarn', 'npm']}>
  <Tab value="bun">bun add @repo/roadmap-ui</Tab>
  <Tab value="pnpm">pnpm add @repo/roadmap-ui</Tab>
  <Tab value="yarn">yarn add @repo/roadmap-ui</Tab>
  <Tab value="npm">npm install @repo/roadmap-ui</Tab>
</Tabs>
