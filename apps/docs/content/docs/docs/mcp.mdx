Kibo UI supports the Model Context Protocol (MCP) for model-driven development.

## Installation

### Cursor

Copy and paste the code into `.cursor/mcp.json`:

```json
{
  "mcpServers": {
    "shadcn": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://www.kibo-ui.com/api/mcp/http"
      ]
    }
  }
}
```

### Windsurf

Copy and paste the code into `.codeium/windsurf/mcp_config.json`:

{
  "mcpServers": {
    "shadcn": {
      "command": "npx",
