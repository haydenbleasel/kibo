## Installation

```package-install
npm i @roadmap-ui/list
```

## Features

- Drag and drop items between groups
- Customize the item contents

## Subcomponents

The list view is made up of the following subcomponents:

| Subcomponent | Description |
| ------- | ----------- |
| Provider | The provider of the list view. |
| Group | A group of items. |
| Item | A single item in the list. |

## Usage

### Basic Example

This example shows a basic list view with a list of features.

<div className="not-prose w-full bg-white border rounded-xl overflow-auto" style={{ height: '32rem' }}>
  <List.Basic />
</div>

```tsx
<List.ListProvider onDragEnd={handleDragEnd}>
  {exampleStatuses.map((status) => (
    <List.ListGroup
      key={status.name}
      id={status.name}
      name={status.name}
      color={status.color}
    >
      {features
        .filter((feature) => feature.status.name === status.name)
        .map((feature, index) => (
          <List.ListItem
            key={feature.id}
            id={feature.id}
            name={feature.name}
            parent={feature.status.name}
            index={index}
          />
        ))}
    </List.ListGroup>
  ))}
</List.ListProvider>
```

### Custom Example

This example shows a list view with a list of features, where each feature has a custom rendering.

<div className="not-prose w-full bg-white border rounded-xl overflow-auto" style={{ height: '32rem' }}>
  <List.Custom />
</div>

```tsx
<List.ListProvider onDragEnd={handleDragEnd}>
  {exampleStatuses.map((status) => (
    <List.ListGroup
      key={status.name}
      id={status.name}
      name={status.name}
      color={status.color}
    >
      {features
        .filter((feature) => feature.status.name === status.name)
        .map((feature, index) => (
          <List.ListItem
            key={feature.id}
            id={feature.id}
            name={feature.name}
            parent={feature.status.name}
            index={index}
          >
            <div
              className="h-2 w-2 shrink-0 rounded-full"
              style={{ backgroundColor: feature.status.color }}
            />
            <p className="m-0 font-medium text-sm flex-1">{feature.name}</p>
            {feature.owner && (
              <Avatar className="h-4 w-4 shrink-0">
                <AvatarImage src={feature.owner.image} />
                <AvatarFallback>
                  {feature.owner.name?.slice(0, 2)}
                </AvatarFallback>
              </Avatar>
            )}
          </List.ListItem>
        ))}
    </List.ListGroup>
  ))}
</List.ListProvider>
```
