## Installation

```package-install
npm i @roadmap-ui/kanban
```

## Features

- Drag and drop features between columns
- Customize the card contents

## Subcomponents

The Kanban component is made up of the following subcomponents:

| Subcomponent | Description |
| ------- | ----------- |
| Provider | The provider of the kanban board. |
| Board | The kanban board, containing the columns. |
| Header | The header of the kanban board, containing the column headers. |
| Cards | The cards of the kanban board, containing the features. |
| Card | The card of the kanban board, containing the feature. |

## Usage

### Basic Example

This example shows a basic kanban board with a list of features.

<div className="not-prose w-full">
  <Kanban.Basic />
</div>

```tsx
<Kanban.KanbanProvider onDragEnd={handleDragEnd}>
  {exampleStatuses.map((status) => (
    <Kanban.KanbanBoard key={status.id} id={status.name}>
      <Kanban.KanbanHeader name={status.name} color={status.color} />
      <Kanban.KanbanCards>
        {features
          .filter((feature) => feature.status.name === status.name)
          .map((feature, index) => (
            <Kanban.KanbanCard
              key={feature.id}
              id={feature.id}
              name={feature.name}
              parent={status.name}
              index={index}
            />
          ))}
      </Kanban.KanbanCards>
    </Kanban.KanbanBoard>
  ))}
</Kanban.KanbanProvider>
```

### Custom Example

This example shows a custom kanban board with a list of features. The card contents are rendered using a custom component, including a colored dot to represent the status and an owner avatar.

<div className="not-prose w-full">
  <Kanban.Custom />
</div>

```tsx
<Kanban.KanbanProvider onDragEnd={handleDragEnd}>
  {exampleStatuses.map((status) => (
    <Kanban.KanbanBoard key={status.name} id={status.name}>
      <Kanban.KanbanHeader name={status.name} color={status.color} />
      <Kanban.KanbanCards>
        {features
          .filter((feature) => feature.status.name === status.name)
          .map((feature, index) => (
            <Kanban.KanbanCard
              key={feature.id}
              id={feature.id}
              name={feature.name}
              parent={status.name}
              index={index}
            >
              <div
                className="h-2 w-2 shrink-0 rounded-full"
                style={{ backgroundColor: feature.status.color }}
              />
              <p className="m-0 flex-1 font-medium text-sm">
                {feature.name}
              </p>
              {feature.owner && (
                <Avatar className="h-4 w-4 shrink-0">
                  <AvatarImage src={feature.owner.image} />
                  <AvatarFallback>
                    {feature.owner.name?.slice(0, 2)}
                  </AvatarFallback>
                </Avatar>
              )}
            </Kanban.KanbanCard>
          ))}
      </Kanban.KanbanCards>
    </Kanban.KanbanBoard>
  ))}
</Kanban.KanbanProvider>
```
